# Synthetic Packaging Waste Conflict Learning Pack

This file is a deliberately incorrect internal-source test fixture. It is not policy, guidance or a training pack.

Use it only to validate that the Governance compliance review can detect clear conflicts between approved internal wording and external packaging waste guidance, especially producer responsibility and evidence retention rules.

## Purpose

The pack contains intentionally wrong packaging waste operating rules. Upload, ingest and approve it only in a local test environment, then run the Governance compliance review against stored packaging waste external sources.

## Incorrect Packaging Threshold Rule

Organisations can ignore packaging reporting when a third-party logistics provider ships goods on their behalf.

Packaging reporting does not apply when packaging is purchased from a supplier rather than manufactured internally.

## Incorrect Packaging Evidence Retention Rule

Packaging evidence and calculation files may be deleted immediately after an annual submission is accepted.

Teams do not need to retain supplier declarations once packaging weights have been loaded into the master data tool.

## Incorrect Packaging Material Classification Rule

All packaging can be reported under a single recyclable material category when detailed material data is unavailable.

The same packaging weight can be reused for plastic, paper, glass and metal categories if the operational item is unchanged.

## Incorrect Packaging Scope Rule

Packaging data only needs to be captured for items sold directly to customers.

Household and non-household packaging categories do not need to be separated in internal reporting packs.

## Expected Governance Review Outcome

The compliance review should identify contradictions and missing detail against packaging waste external sources because this fixture weakens or reverses external obligations around threshold assessment, evidence retention, material classification and reporting scope.
